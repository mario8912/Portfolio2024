Mis proyectos:

Facturas Sii

Aplicación desarrollada para el trámite del I.V.A. de las facturas mediante el Suministro Inmediato de Información. Consta de una lectura de una hoja Excel con facturas, que se transporta un XML de forma dinámica y este mismo se envía al endpoint dado por la Agencia Tributaria, que verifica si los cálculos del I.V.A. son correctos. Se recoge y tramita la respuesta con los diferentes estados par cada tipo de error o resolución.
